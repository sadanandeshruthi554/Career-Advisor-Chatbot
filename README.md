### Career Advisor Chatbot

- An AI-powered Career Advisor Chatbot that helps users explore suitable career paths based on their interests, skills, and goals. This application leverages the Google Gemini API to generate intelligent, human-like responses.
-------------------------------
### 🚀 Live Demo

- https://huggingface.co/spaces/SadanandeShruthi/career-advisor-chatbot
-----------------------------------
### 📌 Features

- 🤖 AI-based career guidance
- 💬 Interactive chatbot interface
- ⚡ Fast and responsive responses
- 🧠 Uses advanced Gemini AI model
- 🌐 Deployed on Hugging Face Spaces
- 🧩 Simple and clean UI with Gradio
----------------------
### 🛠️ Tech Stack

- Programming Language: Python
- Frontend/UI: Gradio
- Backend: Python
- AI Model: Google Gemini API
- Deployment: Hugging Face Spaces

### ⚙️ Installation & Setup

### 1️⃣ Clone the repository
- git clone https://github.com/your-username/career-advisor-chatbot.git
- cd career-advisor-chatbot
------------------
### 2️⃣ Create virtual environment (optional but recommended)

- python -m venv venv
- venv\Scripts\activate   # For Windows
--------------
### 3️⃣ Install dependencies

- pip install -r requirements.txt
--------------------
### 4️⃣ Set up environment variables

- Create a .env file in the root folder and add:
- GOOGLE_API_KEY=your_api_key_here
---------------------------
### 5️⃣ Run the application

- python app.py
-----------------------------
### 🧠 How It Works
1. User enters a career-related question
2. The query is sent to the Gemini API
3. The AI processes and generates a response
4. The chatbot displays the response in real time
-------------------------
